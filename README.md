# LoyaltyMatrix

LoyaltyMatrix is a lightweight loyalty and rewards tracking app designed to help businesses understand, reward, and retain their best customers. It focuses on **simple, data-driven insights** instead of bloated CRM complexity.

## Features

- Customer profiles with basic contact and activity info  
- Points-based loyalty system (earn, redeem, and expire points)  
- Tiered membership levels (e.g. Silver, Gold, Platinum)  
- Activity / transaction history per customer  
- Simple analytics for retention and engagement  
- Admin interface for managing customers and rewards

> Note: This project is under active development. Features and APIs may change frequently.
